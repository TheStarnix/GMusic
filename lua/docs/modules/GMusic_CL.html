GLocalMusic.IsCreated ()
Function that return if the music is created or not by using the GLocalMusic.CurrentAudio variable.

Returns:

    boolean (true if the music is created, false if not)
GLocalMusic.IsValidSong ()
Function that return if the music is valid or not by using the GLocalMusic.CurrentAudio.audioChannel variable.

Returns:

    boolean (true if the music is valid, false if not)
GLocalMusic.Stop ()
Function that change the state of the music. (Playing/Stopped=>DETROYED)

Returns:

    boolean (true if the music has been played/stopped, false if not)
GLocalMusic.SetPause (state)
Function that change the state of the music. (Resume/Pause)

Parameters:

  • state boolean (true if the music will be paused, false if it will be resumed)

Returns:

    boolean (true if the music has been paused/resumed, false if not)
